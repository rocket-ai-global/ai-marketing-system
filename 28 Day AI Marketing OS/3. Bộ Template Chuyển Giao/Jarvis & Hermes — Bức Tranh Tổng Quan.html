Hermes Là Gì? Jarvis Là Gì?

Một câu duy nhất: Hermes là cơ thể — Jarvis là vai trò được trao cho cơ thể đó.

⚙️

Hermes Agent

Engine — Lớp động cơ
  • Mã nguồn mở, MCP-native (hỗ trợ giao thức MCP sẵn) — không bị khoá vào nhà cung cấp nào
  • Chạy 24/7 trên máy chủ ảo (VPS) hoặc máy tính luôn bật
  • Là "cơ thể" cho mọi agent — 1 engine chạy được cả đội nhân viên AI
  • Kết nối thế giới ngoài qua MCP tools: HeyGen (video), API các kênh…
  • Cài 1 lần ở Ngày 0 — gói OS Installed thì đội Rocket cài sẵn
🤵

Jarvis

Persona — Vai trò điều hành
  • Là một Hermes Agent được nạp Master Instruction (bản mô tả nhiệm vụ gốc) vai điều phối
  • Nhận lệnh CEO bằng ngôn ngữ tự nhiên qua Telegram / Zalo
  • Việc nhỏ tự làm — việc chuyên môn giao đúng agent rồi tổng hợp trả CEO trong 1 tin nhắn
  • Chủ động báo cáo: Morning Brief 8h sáng · Evening Report 8h tối
  • Trưởng thành dần trong 28 ngày: sơ sinh → giám đốc điều hành

Lấy API key DeepSeek & khai vào file .env Tự làm được ngay

Vì sao DeepSeek?

💸 Rẻchi phí thấp hơn nhiều lần các model phương Tây — SME chạy 24/7 không xót ví
🔌 Chuẩn OpenAIAPI tương thích chuẩn OpenAI — Hermes cắm vào chạy ngay
🧠 2 chế độdeepseek-chat cho việc thường ngày · deepseek-reasoner khi cần suy luận sâu

Các bước lấy key:

  • Vào platform.deepseek.com → đăng nhập → menu API Keys
  • Bấm Create new API key → đặt tên (VD: jarvis-congty) → copy key ngay (chỉ hiện 1 lần duy nhất)
  • Vào mục Billing nạp credit trước khi dùng

Khai vào file .env trên máy chạy Hermes:

# .env — cấu hình LLM (model ngôn ngữ lớn) LLM_PROVIDER=deepseek LLM_BASE_URL=https://api.deepseek.com LLM_MODEL=deepseek-chat # việc thường ngày; đổi deepseek-reasoner khi cần suy luận sâu DEEPSEEK_API_KEY=sk-xxxxxxxxxxxxxxxx # ← dán key vừa copy
🚫 API key CHỈ nằm trong file .env — tuyệt đối không dán vào note/vault, không chụp màn hình gửi nhóm chat (quy tắc trong CLAUDE.md của Vault SME).

4b. Khai token vào config và giới hạn chỉ mình bạn được chat với bot:

# hermes.config — phần gateway Telegram gateway: type: telegram bot_token: "7123456789:AAHxK9...zQw" # ← token từ BotFather allowed_users: ["123456789"] # ← Telegram ID của CEO (lấy bằng cách chat với @userinfobot)

4c. Khởi động lại Hermes → mở bot của bạn → bấm Start → nhắn tin đầu tiên:

Chào Jarvis!
🤵 Chào sếp! Em là Jarvis — trợ lý điều hành của sếp. Em đã kết nối với vault, sẵn sàng nhận việc ạ.
🚫 Token bot = chìa khoá nhà: lộ token là người khác điều khiển được bot. Nếu lỡ lộ → chat @BotFather gõ /revoke để cấp token mới.
✅ Luôn khai allowed_users — không giới hạn thì bất kỳ ai tìm ra bot đều chat được với "não" doanh nghiệp của bạn.
