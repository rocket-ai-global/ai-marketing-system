# Template 1.1 — Business Snapshot & Business Model Canvas

> Mục tiêu: giúp học viên mô tả doanh nghiệp trong 1 trang và nhìn ra mô hình kinh doanh trước khi đưa AI vào.

## A. Business Snapshot — bản chụp 1 trang
| Trường | Nội dung cần điền | Gợi ý |
|---|---|---|
| Doanh nghiệp |  | Tên, ngành, địa bàn |
| Khách hàng chính |  | 1-2 phân khúc trả tiền thật |
| Sản phẩm/offer chính |  | Tên, giá, kết quả/kỳ vọng |
| Kênh bán hiện tại |  | Facebook/Zalo/Website/Offline/Referral... |
| Điểm nghẽn lớn nhất |  | Lead, chốt, chăm sóc, vận hành, dữ liệu |
| Mục tiêu 28 ngày |  | 3 chỉ số: hiện tại → mục tiêu → cách đo |
| Người phụ trách |  | Ai chịu trách nhiệm triển khai AIOS |

## B. BMC 9 khối
| Khối | Câu hỏi | Nội dung |
|---|---|---|
| Phân khúc khách hàng | Ai trả tiền chính? |  |
| Giá trị cốt lõi | Khách mua vì kết quả gì? |  |
| Kênh | Khách biết/mua/nhận hỗ trợ ở đâu? |  |
| Quan hệ khách hàng | Tư vấn/chăm sóc bằng gì? |  |
| Dòng doanh thu | Tiền đến từ gói/sản phẩm nào? |  |
| Nguồn lực chính | Dữ liệu/người/tài sản nào quan trọng? |  |
| Hoạt động chính | Việc nào tạo ra doanh thu? |  |
| Đối tác chính | Ai giúp giao hàng/bán hàng? |  |
| Cơ cấu chi phí | Chi phí lớn nhất là gì? |  |

## C. Kiểm tra đạt
- [ ] Người ngoài đọc 60 giây hiểu doanh nghiệp bán gì, cho ai, khác gì.
- [ ] Có ít nhất 3 con số thật.
- [ ] Mục tiêu 28 ngày đo được.
- [ ] Không viết kiểu “mọi người đều là khách hàng”.
