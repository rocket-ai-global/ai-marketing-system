QUY TRÌNH TẠO HỒ SƠ KHÁCH HÀNG CHUẨN — 5 BƯỚC
================================================

Bộ tài liệu này gồm 3 file:

1. Quy Trình Tạo Hồ Sơ Khách Hàng — Canvas.canvas
   → Bản đồ trực quan 5 bước của quy trình (mở bằng Obsidian).

2. HSKH mẫu.pdf
   → File mẫu hồ sơ khách hàng — dùng ở BƯỚC ① (đưa vào Claude làm khuôn mẫu).

3. Phiếu điền hồ sơ doanh nghiệp.pdf
   → Phiếu khai thông tin thương hiệu — dùng ở BƯỚC ② (điền xong đưa vào Claude).


CÁCH MỞ FILE CANVAS
-------------------
- File .canvas chỉ mở được trong Obsidian (miễn phí: https://obsidian.md).
- Cách dùng: mở Obsidian → kéo cả 3 file này vào cùng một vault (thư mục Obsidian)
  → mở file .canvas để xem toàn cảnh 5 bước.
- Nếu để chung 3 file trong cùng vault, các liên kết tới 2 file PDF trong canvas
  sẽ tự nhận diện.


TÓM TẮT 5 BƯỚC
--------------
① Lấy file mẫu hồ sơ (HSKH mẫu.pdf) → đưa vào Claude.
② Chuẩn bị thông tin thương hiệu (điền Phiếu điền hồ sơ doanh nghiệp.pdf).
③ Phân tích đối thủ (mình + 5 đối thủ trực tiếp, theo 13 đầu mục).
④ Tạo hồ sơ theo form chuẩn (bổ sung từng bước một).
⑤ Kiểm tra & bổ sung (bắt Claude tự soi lại theo 7 tiêu chí).
   → Lặp lại ④ → ⑤ đến khi hồ sơ sắc, thật, đời.
