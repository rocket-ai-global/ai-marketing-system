# Checklist Review BMC — Tuần 1

> Dùng checklist này để chấm bài BMC của học viên. Mỗi mục: Đạt / Chưa đạt / Cần bổ sung.

---

## 🔴 PHẦN 1: CHÂN DUNG KHÁCH HÀNG (Quan trọng nhất — 40% điểm)

| # | Tiêu chí | Đạt? |
|---|----------|:---:|
| 1.1 | Xác định được **1 nhóm KH cụ thể** (không phải "tất cả mọi người") | |
| 1.2 | Có **độ tuổi, nghề nghiệp, thu nhập, vị trí** rõ ràng | |
| 1.3 | Liệt kê **3 pain point** (nỗi đau) — cụ thể, không chung chung | |
| 1.4 | Liệt kê **3 desire** (mong muốn) — họ muốn đạt được gì? | |
| 1.5 | Liệt kê **3 fear** (nỗi sợ) — họ sợ gì khi mua hàng? | |
| 1.6 | Liệt kê **3 objection** (phản đối) — họ thường từ chối vì lý do gì? | |
| 1.7 | Có **Customer Journey** 5-7 giai đoạn (từ chưa biết → mua lại) | |
| 1.8 | Phân biệt được KH theo **mức độ nhận thức** (chưa biết vấn đề / biết vấn đề / biết giải pháp / biết sản phẩm) | |

### Lỗi thường gặp → Cách sửa

| Lỗi | Sửa |
|-----|-----|
| "Khách hàng là nữ 25-45 tuổi" | Quá chung → "Nữ chủ spa 30-40 tuổi, doanh thu 200-500tr/tháng, có 5-10 nhân viên" |
| "Họ muốn đẹp hơn" | Quá mơ hồ → "Họ muốn giảm 3kg trong 2 tuần trước đám cưới con gái" |
| Không có objection | Luôn có — "Đắt quá", "Để em suy nghĩ", "Có chắc hiệu quả không?" |

---

## 🟠 PHẦN 2: OFFER & ĐỊNH VỊ (30% điểm)

| # | Tiêu chí | Đạt? |
|---|----------|:---:|
| 2.1 | Offer có **tên cụ thể** (không phải "gói dịch vụ") | |
| 2.2 | Có **lời hứa chính** — KH đạt được gì, trong bao lâu? | |
| 2.3 | Có **USP rõ ràng** — khác biệt với đối thủ là gì? | |
| 2.4 | Có **Bonus Stack** (quà tặng, hỗ trợ thêm) — tăng giá trị offer | |
| 2.5 | Có **Risk Reversal** (cam kết, bảo hành, hoàn tiền) | |
| 2.6 | Giá bán được **niêm yết rõ ràng** (không "liên hệ để biết giá") | |

---

## 🟡 PHẦN 3: DỮ LIỆU DOANH NGHIỆP (20% điểm)

| # | Tiêu chí | Đạt? |
|---|----------|:---:|
| 3.1 | Có **Tầm nhìn / Sứ mệnh** rõ ràng | |
| 3.2 | Có **Giá trị cốt lõi** (3-5 giá trị) | |
| 3.3 | Có **danh sách sản phẩm/dịch vụ** kèm giá | |
| 3.4 | Có **FAQ** (ít nhất 10 câu hỏi thường gặp) | |
| 3.5 | Có **Case Study / Testimonial** (nếu có) | |
| 3.6 | Có **quy trình bán hàng hiện tại** (các bước từ lead → chốt) | |

---

## 🟢 PHẦN 4: CƠ HỘI AI (10% điểm)

| # | Tiêu chí | Đạt? |
|---|----------|:---:|
| 4.1 | Liệt kê được **10 đầu việc có thể AI hoá** | |
| 4.2 | Chọn ra **Top 3 ưu tiên nhất** (tác động lớn + dễ làm) | |
| 4.3 | Có **mục tiêu 28 ngày** rõ ràng (con số cụ thể) | |

---

## 📊 Cách chấm điểm

| Mức | Điều kiện |
|-----|----------|
| 🟢 **ĐẠT** | ≥80% tiêu chí (18/23) |
| 🟡 **CẦN BỔ SUNG** | 50-79% (12-17/23) |
| 🔴 **LÀM LẠI** | <50% (<12/23) |
