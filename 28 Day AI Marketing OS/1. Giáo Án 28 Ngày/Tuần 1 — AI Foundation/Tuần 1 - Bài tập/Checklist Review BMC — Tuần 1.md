# Checklist Review BMC — Tuần 1

> Dùng checklist này chấm bài BMC của học viên. **2 case mẫu:** 🍵 MSX Group (B2C — trà) + 🚀 RocketAI (B2B — dịch vụ). Mỗi mục: Đạt / Chưa đạt / Cần bổ sung.

---

## 🔴 PHẦN 1: CHÂN DUNG KHÁCH HÀNG (40% điểm)

| # | Tiêu chí | Đạt? |
|---|----------|:---:|
| 1.1 | Xác định **1 nhóm KH cụ thể** (không phải "tất cả mọi người") | |
| 1.2 | Có **độ tuổi, nghề nghiệp, thu nhập, vị trí** rõ ràng | |
| 1.3 | Liệt kê **3 pain point** cụ thể — viết bằng ngôn ngữ KH | |
| 1.4 | Liệt kê **3 desire** — họ muốn đạt được gì? | |
| 1.5 | Liệt kê **3 fear** — họ sợ gì khi mua hàng? | |
| 1.6 | Liệt kê **3 objection** — họ từ chối vì lý do gì? | |
| 1.7 | Có **Customer Journey** 7 giai đoạn | |
| 1.8 | Phân biệt KH theo **mức độ nhận thức** | |

### Lỗi thường gặp → Cách sửa

| ❌ Sai | ✅ Đúng | Tham khảo |
|--------|---------|-----------|
| "Khách hàng là nữ 25-45" | "Chị Hương 40t, văn phòng Hà Nội, quan tâm sức khoẻ, mua trà online" | [[Case Study Mẫu — RocketAI & MSX Group]] → MSX |
| "Muốn đẹp hơn, khoẻ hơn" | "Muốn ngủ ngon tự nhiên không cần thuốc, muốn SP có nguồn gốc rõ ràng" | MSX: Chị Hương muốn trà thảo mộc giúp ngủ ngon |
| Không có objection | Luôn có: "Đắt quá", "Có khác gì hàng chợ?", "Uống bao lâu mới có tác dụng?" | MSX FAQ 12 câu |
| KH "tất cả mọi người" | KH mua trà: nữ 30-50, quan tâm sức khoẻ. KH mua AI: chủ DN nữ 30-45 | So sánh MSX (B2C) vs RocketAI (B2B) |

## 🟠 PHẦN 2: OFFER & ĐỊNH VỊ (30% điểm)

| # | Tiêu chí | Đạt? |
|---|----------|:---:|
| 2.1 | Offer có **tên cụ thể** | |
| 2.2 | Có **lời hứa chính** — KH đạt được gì, trong bao lâu? | |
| 2.3 | Có **USP rõ ràng** — khác biệt với đối thủ | |
| 2.4 | Có **Bonus Stack** — tăng giá trị offer | |
| 2.5 | Có **Risk Reversal** (cam kết, hoàn tiền) | |
| 2.6 | Giá **niêm yết rõ ràng** — không "liên hệ để biết giá" | |

---

## 🟡 PHẦN 3: DỮ LIỆU DOANH NGHIỆP (20% điểm)

| # | Tiêu chí | Đạt? |
|---|----------|:---:|
| 3.1 | Có **Tầm nhìn / Sứ mệnh** rõ ràng | |
| 3.2 | Có **Giá trị cốt lõi** (3-5 giá trị) | |
| 3.3 | Có **danh sách sản phẩm/dịch vụ** kèm giá | |
| 3.4 | Có **FAQ** (ít nhất 12 câu — như MSX) | |
| 3.5 | Có **Case Study / Testimonial** (nếu có) | |
| 3.6 | Có **quy trình bán hàng hiện tại** | |

| 4.3 | Có **mục tiêu 28 ngày** rõ ràng (con số cụ thể) | |
