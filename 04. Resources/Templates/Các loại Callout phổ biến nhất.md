**Các loại callout phổ biến và màu tương ứng:**

| Cú pháp      | Màu        |
| ------------ | ---------- |
| `[!note]`    | Xanh dương |
| `[!tip]`     | Xanh lá    |
| `[!warning]` | Vàng cam   |
| `[!danger]`  | Đỏ         |
| `[!info]`    | Xanh nhạt  |
| `[!quote]`   | Xám        |

---

Chỉ cần gõ `> [!tip]` rồi Enter là Obsidian tự render (hiển thị) ra khung màu đó luôn — không cần cài thêm plugin gì.
