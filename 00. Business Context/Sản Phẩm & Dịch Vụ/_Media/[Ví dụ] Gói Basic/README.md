# Media — [Ví dụ] Gói Basic

Thư mục media của [[[Ví dụ] Gói Basic]]. Bỏ hình vào `hinh/`, video vào `video/` rồi nhúng vào hồ sơ sản phẩm bằng `![[...]]`. Xoá thư mục ví dụ này khi bắt đầu làm sản phẩm thật.

```
hinh/   ← ảnh sản phẩm, ảnh gói, infographic bảng giá
video/  ← video demo / hướng dẫn / giới thiệu
```
